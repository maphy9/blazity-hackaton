# Project Vocabulary

Use this file to define canonical product and codebase terms for AI agents.

## Terms

| Term | Meaning | Avoid |
| --- | --- | --- |
| Brief | The single source idea/content the user writes once, before adaptation. | prompt (reserve "prompt" for the LLM input) |
| Platform | A target channel the content is published to (LinkedIn, Instagram, X/Twitter, newsletter). | network, site |
| Adaptation | AI-generated version of the brief tailored to one platform's tone, length, and format. | rewrite, translation |
| Connector | Integration that publishes an adaptation to a platform using user-supplied API credentials. | integration (be specific), publisher |
| One Idea, Every Format | The product tagline/concept: write once, generate for every selected platform. | — |
